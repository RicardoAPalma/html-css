Usando fontes externas baixadas

  1. Baixe a fonte e jogue na pasta que deseja.

  2. No vscode, dentro da tag <style> acrescente outra regra e clique na segunda opção.

    3. @font-face {

    font-family: ;

    src: url()

    }

  3. Depois de font-family: deve ser criado um "apelido" para a fonte.

    4. font-family:'fonte';


  4. Dentro dos parenteses deve ser colocado o nome da fonte que foi baixada exatamente como ela está na pasta, no meu caso foi a "Sketching universe".


src: url('fonts/Sketching Universe.otf')


Como ela está dentro de uma pasta, chamada "fonte" que está dentro de outra pasta, devemos colocar o "/"


Quando for usar a fonte que foi baixada, em font-family deve ser colocado o "apelido" que foi craido, que nesse caso foi o "fonte"

Exemplo:

h1{

pfont-family: 'fonte';

font-size: 3em;

font-weight: normal;

}

Tipos e formatos de fontes

  • Opentype (normamente usado quando é formato otf)

  • Truetype (normalmente usado quando é formato ttf)

  • Embedded-opentype

  • Truetype -aat (Apple Advanced Typography)

  • Svg

No nosso caso o formato que foi utilizado foi o opentype
